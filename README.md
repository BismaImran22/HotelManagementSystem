# HotelManagementSystem
Hotel Management System – Project Description (GitHub Ready) Overview  The Hotel Management System is a desktop-based application designed to automate and simplify the daily operations of a hotel. The system replaces manual paperwork and improves staff efficiency by providing a centralized platform for managing rooms, guests, bookings, billing
Purpose

The main purpose of this project is to streamline hotel operations by reducing human errors, saving time, and improving the overall guest experience. It provides quick access to real-time room availability, customer information, and financial reports.

Features
✔ 1. Authentication & User Roles

Secure login system

Admin and Receptionist accounts

✔ 2. Dashboard

Overview of hotel status

Quick-access navigation

✔ 3. Room Management

Add, update, and delete rooms

Track room status (Available, Occupied, Reserved, Dirty)

Set room price and category

✔ 4. Guest/Customer Management

Store and manage guest information

View booking and stay history

✔ 5. Booking & Reservation System

Make new reservations

Modify or cancel reservations

Real-time room availability

✔ 6. Check-In / Check-Out

Automatic room allocation

Update room status

Generate check-out summary

✔ 7. Billing & Invoice Generation

Automatic calculation of charges

Add taxes, discounts, and extra services

Generate and print invoices

✔ 8. Reports

Daily revenue report

Room occupancy report

Guest history report

Scope

This system is designed for small to medium-sized hotels. It supports:

Local desktop-based operations

File/Excel-based data storage or simple database integration

Single-hotel management

Not included: online booking website, restaurant POS, or chain hotel management.
